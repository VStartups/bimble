About Bimble

The Bimble Programming Language combines the simplicity of Python, the speed of Rust, and execution similar to Java. It targets general-purpose programmers who want to create simple GUI (future plans) and CLI applications. Bimble requires the Bimble Virtual Machine (BVM) to run.
